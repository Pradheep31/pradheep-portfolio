Projects

Thiraipadam - Movie Platform (Group Project)

Thiraipadam is a collaborative project where we built a movie-watching platform focused on clean visuals and easy navigation. I was responsible for designing the movie card layout, navigation structure, and enhancing the visual styling. This project taught me how to share responsibilities in a team and maintain design consistency across different sections. We worked together to ensure a polished and intuitive end-user experience.

Strikingly Website Clone

This project is a pixel-perfect clone of the Strikingly homepage, developed using HTML5, CSS3, and Bootstrap. It helped me deeply understand the importance of layout structure, spacing, typography, and responsive design. Special focus was given to mimicking the real site's user experience and aesthetic appeal, making this a valuable exercise in visual accuracy and attention to detail.
